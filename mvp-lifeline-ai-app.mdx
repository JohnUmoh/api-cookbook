## 🌍 MVP LifeLine AI App

MVP LifeLine is a **global AI-powered mobile platform** designed to help youth **Earn, Heal, and Grow**—regardless of connectivity or location. It combines multilingual support, career tools, mental wellness, and emotional tracking—all guided by Perplexity's powerful **Sonar API** for natural, dynamic interactions.

---

## 🚀 Key Features

---

### 🔍 Core Functional Zones & Modules

1. 📁 **Career Zone**  
Smart CV builder, AI job matching, real-time interview coaching.  
Powered by Sonar for job market insights & personalized tips.

2. 💔 **Emotional Zone**  
Sentiment-aware journaling.  
AI-based emotional check-ins & motivational prompts.

3. 🤖 **SmartQ Access**  
Ask anything, anytime.  
Sonar-powered queries (career, emotional, daily questions).

4. 🧠 **Dual Mode AI**  
Switch between Career Guide and Emotional Support.  
Context-aware interaction for holistic help.

5. 💼 **Digital Hustle Hub**  
Freelancing tools, gig economy jobs, passive income ideas.  
Use AI to find, filter, and pitch gigs.

6. 🤝 **Partner Zone**  
NGO, mentor, and impact partner ecosystem.  
AI-curated match based on user's goals.

7. ✅ **Productivity Zone**  
AI-powered Kanban board, daily planners, habit tracking.

8. 🧘 **Wellness Zone**  
Calming sounds, meditations, daily mental check-ins.

9. 💰 **Finance Zone**  
Budget tracker, savings coach, money tips for youth.

10. 📊 **Insight Zone**  
Analytics dashboard for usage, growth, and behavior trends.

11. 🔄 **Sync Zone**  
Offline data sync via SMS/USSD integration.

---

## 🧠 How We Used the Sonar API

- **SmartQ Agent**: Built an intelligent multilingual assistant powered by Sonar API to understand emotional tone, answer user questions, and guide users in real time.
- **Prompt Optimization**: Customized Sonar prompts for emotional tracking, CV feedback, and gig search automation.
- **Offline-first architecture**: Designed the backend to store prompt/response data for re-syncing when internet is restored.

## 🎯 Impact & Vision

MVP LifeLine began in Africa, but is built for the world. With the UN reporting that over **1.3 billion youth** face career and mental health barriers, especially in emerging markets, we created this app to:

- Break language and data access barriers  
- Offer offline AI support for remote communities  
- Integrate mental wellness with productivity and earning tools  
- Give youth dignity, digital skills, and hope through tech
